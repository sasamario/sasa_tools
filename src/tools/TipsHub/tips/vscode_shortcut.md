| ショートカット | コマンド詳細 |
|------------|------------|
| `Ctrl + D` | 同じ文字列を順番に選択 |
| `Ctrl + Shift + L` | 選択中の文字列と同じ文字列をすべて選択する |
| `Shift + Alt + I` | 選択した各行の末尾にカーソルを配置する |
| `Ctrl + Alt + ↑ / ↓` | カーソルを上下に追加する |
| `Alt + クリック` | 任意の位置にカーソルを追加する
