| ショートカット | コマンド詳細 |
| --- | --- |
| Win + Shift + S | 画面の一部または全体をキャプチャする（スクリーンショット） |
| Win + G | Xbox Game Barを起動する。画面録画やパフォーマンス確認が可能 |
| Win + E | エクスプローラーを開く |
| Win + D | デスクトップを表示／元に戻す |
| Ctrl + X | 切り取り |
| Ctrl + Z | 操作を元に戻す |
| Ctrl + Y | 元に戻した操作をやり直す |
| Win + Tab | タスクビューを表示し、仮想デスクトップを管理する |
| Alt + Tab | 起動中のアプリを切り替える |
| Ctrl + Shift + Esc | タスクマネージャーを直接起動する |
| Win + ← / → | ウィンドウを左右半分に配置する |
| Win + ↑ | ウィンドウを最大化する |
| Win + ↓ | ウィンドウを最小化する |
| Win + . | 絵文字・顔文字パネルを開く |
| Win + V | クリップボード履歴を表示する |
| Ctrl + Shift + N | 新しいフォルダを作成する |
